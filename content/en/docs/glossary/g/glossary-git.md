title: "Git"
linkTitle: "Git"
description: >
  _Git_ is a free and open source distributed version control system.
  It is used for controlling, organizing, and tracking different versions in history of computer files, typically text files but also any other type of file.
  It allows all changes made to a file to be viewed and tracked over time and for branching/merging of the repository for separate strands of work.
---

The source code for the Stroom software is stored in a Git repository.
Stroom also uses Git for managing user content that is held in one or more Git repositories.
