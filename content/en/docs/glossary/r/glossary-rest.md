description: >
  _REST_ (**Re**presentational **S**tate **T**ransfer) is essentially an architectural style that dictates how data should be handled and "transferred" across a network.
  REST APIs typically use _JSON_ to send data between the client and the server, and the HTTP methods GET, PUT, PATCH, POST and DELETE.
