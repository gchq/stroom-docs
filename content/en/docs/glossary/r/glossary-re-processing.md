---
title: "Re-processing"
linkTitle: "Re-processing"
description: >
  The act of repeating the processing of a set of input data (_Stream_) that have already been processed at least once.
  Re-Processing can be done for an individual Stream or multiple Streams using a _Processor Filter_.
