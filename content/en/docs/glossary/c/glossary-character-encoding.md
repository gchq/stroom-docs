---
title: "Character encoding"
linkTitle: "Character encoding"
description: >
  _Character Encoding_ is the means of encoding character data (i.e. text) into binary form.
  Therefore to decode character data from a stream of bytes, the character encoding must be known (or guessed).
---

Common examples of character encodings are `ASCII`, `UTF-8` and `UTF-16`.

Each {{< glossary "feed" >}} has a defined character encoding for the data and {{< glossary "context data" "context">}}.
This allows Stroom to decode the data sent into that Feed.
