A Query document is used to run ad-hoc searches against a single data source such as a [Lucene Index]({{< relref "/docs/reference-section/documents#lucene-index" >}}), [View]({{< relref "/docs/reference-section/documents#view" >}}) or [Searchable]({{< relref "/docs/user-guide/search/data-sources" >}}).
Queries are written using the [{{% glossary "StroomQL" %}}]({{< relref "stroom-query-language" >}}) query language.

For multi-source or visualisation-heavy use cases, consider using a [Dashboard]({{< relref "../dashboards" >}}) instead.
