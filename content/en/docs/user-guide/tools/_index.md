Stroom provides a number of supplementary tools for administrative and data access tasks.

* [Command Line Tools]({{< relref "command-line" >}}) — a set of CLI commands for tasks such as database migration, managing users, and importing/exporting content. Run from the server using `java -jar stroom-app-all.jar <command>`.
* [Viewing Data]({{< relref "../viewing-data" >}}) — how to view and navigate raw and processed data within Stroom's stream store.
