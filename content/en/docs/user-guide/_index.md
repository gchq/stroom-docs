This section is the primary reference for using Stroom day-to-day.
It covers all aspects of operating Stroom, from searching and indexing data to managing pipelines, security and system configuration.

Key areas of the user guide include:

* [Pipelines]({{< relref "pipelines" >}}) — building data ingest, transformation and indexing pipelines.
* [Searching Data]({{< relref "search" >}}) — querying data using Dashboards, Queries and Analytic Rules.
* [Indexing Data]({{< relref "indexing" >}}) — indexing event data for fast retrieval using Lucene, Elasticsearch or Solr.
* [Content Management]({{< relref "content" >}}) — creating, organising and managing Stroom content items.
* [Security]({{< relref "security" >}}) — managing user accounts, groups and permissions.
* [Data Receipt]({{< relref "data-receipt" >}}) — controlling how Stroom receives and processes incoming data.
* [Background Jobs]({{< relref "jobs" >}}) — managing Stroom's background processing jobs.
* [API]({{< relref "api" >}}) — using Stroom's REST APIs.
* [Tools]({{< relref "tools" >}}) — command-line tools for administering Stroom.
